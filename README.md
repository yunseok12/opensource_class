# 양윤석 자기소개 페이지

## 소개
HTML, CSS, JS를 활용한 포트폴리오 페이지입니다. 관심사, 목표, 좋아하는 것들을 소개합니다.

## 실행 방법
GitHub Pages를 통해 배포된 링크에서 바로 확인 가능합니다.

## 파일 구조
- index.html
- css/
  - reset.css
  - style.css
- js/
  - loading.js
  - main.js
- images/

## 주요 기능
- 로딩 애니메이션
- 이미지 슬라이드
- 반응형 디자인 및 인터랙션 애니메이션
